```
模型在ios端无法进行触摸旋转

    在加载模型的canvas 容器中，加个属性 touch-action=none;即可，(但是该属性在安卓端会导致模型不可旋转，可引入pep.min.js解决)
```
 
```
3d模型嵌套（透过外部模型看到内部模型）

    只需要设置mesh.renderingGroupId（如mesh.renderingGroupId=99;），另外提示：即便修改了这个属性，如果您用射线拾取场景中的模
    型做击中效果，射线击中反馈的数组中，还是按照正常的场景进行深度计算，也就是说如果需要透视显示的模型在场景中设置了mesh.renderOrder=99，
    同时出现在墙的后面，射线拾取的仍然是墙体，而并非墙后面的模型。
```
